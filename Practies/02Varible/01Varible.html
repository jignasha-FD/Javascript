Variables are Containers for Storing Data


There are two types of variables in JavaScript


1.Local varible

Local variables are created when the function starts its execution and are lost when the function ends.


2.Global varible

Global variables, on the other hand, are created as execution of the program begins and are lost when the program is ended.


JavaScript Variables can be declared in 4 ways:

1.let
2.var
3.const

4.Automatically
